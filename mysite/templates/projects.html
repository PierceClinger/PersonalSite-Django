For our Capstone project we decided to create a cryptocurrency paper trading web application called Coin Trader. This application allows for users to create accounts and play around with trading crypto without the risk of losing your hard earned money. This was so much fun to create and allowed us to work with Django, which was brand new to all of us. The application itself was programmed in python, but also required a front-end which was coded in basic HTML and CSS. There was also some knowledge in SQL required since we needed to store all of our data in the Django database. I specifically worked on the front-end, importing the data from the Coin Base API, storing/accessing the data, creating the graphs, and also implementing background tasks (multiple threads) to allow for data to be imported while users access the app. This was by far my favorite project I have orked on due to having an awesome group and constantly learning new things. While the code may seem a little sloppy, we succeeded in creating the desired application with a framework we had never worked with before.

Skills
